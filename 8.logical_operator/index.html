logical operator

    >> &&(and)
  • if all the value are true then it will give true
  • if one of the value is false then it will give false
  • if the first value is false it will give false it will not go for the next value


    • >>||(or)
  • if all the value is false then it will give false
  • if one of the value is true then it will give true
  • if the first value is true it will give true it will not go for the next value


    • >>!(not)
