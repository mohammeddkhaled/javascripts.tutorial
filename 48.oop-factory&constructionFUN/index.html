OOP and factory functions in JavaScript

four pillars of OOP

  • Encapsulation

    • Encapsulation is the bundling of data and methods that operate on that data within one unit, such as a class in OOP.

  • Abstraction

    • Abstraction is the concept of hiding the complex implementation details and showing only the essential features of the object.

  • Inheritance

    • Inheritance is a mechanism where a new class can inherit properties and methods from an existing class.

  • Polymorphism

    • Polymorphism allows objects of different classes to be treated as objects of a common superclass.
