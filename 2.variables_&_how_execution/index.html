Variable in javascript

A JavaSccripts variable is just a name of storage

And we can give that name in three ways.

  • using letkeyword.
  • using constkeyword
  • using varkeyword

A Variable name can contain any for the folling characters but must not start with a digit.

  • capital letters: A to Z
  • small letters:a to z
  • digits:0 to 9
    • (but we can not start variable name with digit)
  • only two soecial characters that are : "$" "-"
